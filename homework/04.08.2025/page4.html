create an html file

  1. Right-click inside the folder area in VS Code’s sidebar and select New File.

  2. Name it index.html (or another .html name)
  3. Type the HTML boilerplate manually, or type ! and press Tab (Emmet shortcut) to auto-generate it

    1. Add headings, paragraphs, images, or other HTML element

    2. Press Ctrl + S (Windows) or Cmd + S (Mac) to save

  4. Right-click the HTML file in VS Code and select “Reveal in File Explorer/Finder”, then double-click the file to open in your default browser

  5. Install the Live Server extension to preview your changes in real-time
